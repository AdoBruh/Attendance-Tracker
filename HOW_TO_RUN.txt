HOW TO RUN THE LARAVEL ATTENDANCE TRACKER

1. Open XAMPP.
2. Start Apache if you want to use XAMPP, but for this project you can also use Laravel serve.
3. Open Command Prompt or PowerShell.
4. Go to the project folder:

   cd "C:\Users\feria\Documents\Codex\2026-05-30\categories-for-second-table-student-records\AttendanceTracker"

5. Run the Laravel server:

   C:\xampp\php\php.exe artisan serve --host=127.0.0.1 --port=8000

6. Open this in your browser:

   http://127.0.0.1:8000

Register a new account first, then login and use the Attendance Tracker.
